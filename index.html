Opacity

Definition

Opacity is the solidity of an element. It is the amount of visibility of content behind the element. It is the oposite of transparency.

Values

Opacity can be set to either a decimal, 0-1, or a percentage, 0-100%. A value of 0 or 0% specifies that the element is transparent. A value of 1 or 100% specifies that the element is completely opaque.

See the Pen Opacity Values by kandinma (@kandinma) on CodePen.

Acessiblity

Changing the opacity of an element that includes text can effect the readiblity. Keep this in mind as some people may not be able to read the text if opacity is set a curtain way.
